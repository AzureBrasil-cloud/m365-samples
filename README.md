# M365 Samples: Azure Brasil

> A consolidated repository of Microsoft 365 resources, guides, and reference implementations by **Azure Brasil**.

This repository brings together community-driven samples, documentation, and best practices for working with Microsoft 365 (formerly Office 365) and related Microsoft cloud services. It is a knowledge hub for organizations looking to implement, extend, and optimize their Microsoft 365 environments.

---

## 📚 What's Inside

### [🧠 Work IQ](./work-iq/)

A comprehensive guide to **Microsoft Work IQ**, the intelligence layer connecting AI assistants to Microsoft 365 Copilot data.

- **What it is:** Learn about the architecture, capabilities, and security model of Work IQ
- **Setup:** Step-by-step guidance for M365 admins to enable and provision Work IQ in your tenant
- **Installation:** Install and configure the Work IQ CLI for terminal usage
- **Editor Integration:** Use Work IQ with VS Code through the Model Context Protocol (MCP)
- **Examples:** Prompt catalog and real-world usage scenarios

**Get started:** [→ work-iq/README.md](./work-iq/)
